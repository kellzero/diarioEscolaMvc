# ?? Diário Escolar MVC

Aplicação web desenvolvida em **ASP.NET MVC** com o objetivo de gerenciar informações escolares, como alunos, registros e dados acadêmicos, de forma organizada e eficiente.

O projeto segue o padrão **MVC (Model-View-Controller)**, separando responsabilidades e facilitando a manutenção e evolução do sistema.

---

## ?? Funcionalidades

- ? Cadastro e gerenciamento de informações escolares
- ? Organização de dados em uma estrutura MVC
- ? Interface web utilizando Views
- ? Integração entre Controllers e Models
- ? Estrutura preparada para expansão futura
